## 주제 : 8. 웹을 이용한 스노우 필터 카메라 만들기
우수참가자로 선정시 'NAVER'에서 체험형 인턴십의 참여기회가 주어집니다.

### 주제선정 배경
- 웹에서 스노우앱과 같은 실시간 필터링이 가능한 카메라를 구성 필터가 적용된 카메라 내용을 저장한다. 이를 통해 웹에서의 동영상 처리 기술을 확보\

### 개발 요구사항 (필수)
- 웹 개발 환경
- 노트북
- Web 개발에 필요한 소프트웨어

### 개발 요구사항 (선택)
- 웹을 이용하여 실시간 카메라 필터를 적용할수 있다.
- 실시간 필터링을 위한 성능

### Role
- FrontEnd

### Technology

- Canvas
